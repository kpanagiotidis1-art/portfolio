# Kosta Panagiotidis - Portfolio Assessment

A personal portfolio website built to showcase my projects, skills, and development journey.

## Built With
- HTML
- SCSS (BEM methodology)
- JavaScript
- Google Fonts (Raleway, Work Sans)
- Font Awesome
- Formspree

## Features
- Multi-page layout with consistent design system
- Fullscreen overlay navigation with hamburger animation
- Interactive skills section with category filtering
- Project carousel with peek effect
- Working contact form via Formspree
- Fully responsive across mobile, tablet and desktop
- Page transition animations
